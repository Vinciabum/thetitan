  - CTA는 반드시 포함 (내용 요약 + 행동 유도)

## 2. 고성능 카드뉴스 공통 패턴

### 2-1. 슬라이드 구조 공식
```
P1 (커버):    강렬한 후킹 문구 + 시선 집중 비주얼
P2~P(n-1):   핵심 내용 전달 (개념/사례/비교/인사이트)
P(n) (마지막): CTA — 저장/팔로우/댓글/DM 유도
```

### 2-2. 후킹 문구 패턴 (저장/공유율 높은 유형)
| 유형 | 예시 | 효과 |
|------|------|------|
| 선언형 | "2026년 마음가짐 제출합니다" | 감정 동조, 공유 유발 |
| 숫자형 | "Claude 프롬프트 5가지" | 구체성, 클릭 유발 |
| 질문형 | "팬까지 만드는 방법이 궁금하다면?" | 호기심 유발 |
| 부정형 | "너무 많은 색상을 사용하지 마세요" | 반성/경계 자극 |
| 호기심형 | "소름 끼치는 사실들" | 즉각 클릭 유발 |

### 2-3. 텍스트 위계 원칙 (blunge.ai 정리 기반)
1. **크기(Size)**: 제목 > 부제목 > 본문 — 가장 중요한 것이 가장 크게
2. **굵기(Weight)**: Bold는 시선을 끌고, Regular는 부드럽게 읽힘
3. **간격(Spacing)**: 글자 간격, 줄 간격, 블록 간 여백으로 그룹화 명확히
4. **색상(Color)**: 강조 정보에만 포인트 컬러 사용 (1~2색 제한)

### 2-4. 참여율을 높이는 콘텐츠 요소
| 요소 | 설명 |
|------|------|
| 공감 | "내 얘기같다" → 공유 유도 |
| 인사이트 | 몰랐던 정보 → 저장 유도 |
| 감동 | 정서적 울림 → 공유/저장 |
| 유머 | 가볍게 퍼짐 → 도달 확대 |
| 실용 | 바로 써먹을 수 있는 것 → 저장 |

### 2-5. CTA 유형별 효과
```
① 댓글 CTA: "댓글에 'XX'라고 남기세요" → 댓글 수 폭발 (platformtree 156개)
② 저장 CTA: "저장해두고 오늘 밤 꺼내 읽어보세요" → 저장률 1위 신호
③ 팔로우 CTA: "더 많은 팁은 팔로우" → 팔로워 전환
④ 공유 CTA: "이 말이 필요한 사람에게 보내주세요" → 도달 확대
⑤ 개방형 질문: "여러분은 어떠신가요?" → 텍스트 댓글 유도
```

