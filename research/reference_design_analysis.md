## 2. 비주얼 디자인 시스템 상세 분석

### 2-1. 배경 이미지
- **스타일**: 흑백(Black & White) 빈티지/필름 감성 사진
- **소재**: 1950~1960년대 클래식 여성 인물 사진 (오드리 헵번 스타일)
- **구도**: 인물이 주인공, 자연스러운 동작 (독서, 전화, 카메라 등)
- **분위기**: 사색적(contemplative), 조용한 힘(quiet strength), 지적 감성
- **풀블리드**: 이미지가 슬라이드 전체를 채움 (여백 없음)
- **명도**: 전체적으로 어두운 톤 → 흰 텍스트와 강한 대비 형성

### 2-2. 텍스트 배치
```
[슬라이드 레이아웃 구조]

┌─────────────────────────────┐
│ SOCIAL MATURITY    슬라이드#/10 │  ← 상단: 소주제 레이블(영문) + 번호
│                               │
│   [흑백 사진 풀블리드]          │
│                               │
│        대제목 (Bold)           │  ← 사진 중하단: 큰 제목
│   부제목 또는 소제목 (smaller)  │
│                               │
│  본문 텍스트 2~4줄 (Regular)   │  ← 제목 아래: 설명 문장
│                               │
│              ONE STEP AHEAD   │  ← 우측 하단: 브랜드명 (커버만)
└─────────────────────────────┘
```

### 2-3. 타이포그래피
| 요소 | 스펙 |
|------|------|
| 커버 제목 | 한국어 Bold, 대형 (화면 너비 60~70%) |
| 슬라이드 제목 | 한국어 Bold, 중대형 |
| 본문 | 한국어 Regular, 소형, 줄간격 넓음 |
| 상단 레이블 | 영문 CAPS, 매우 작게 (SOCIAL MATURITY) |
| 페이지 번호 | 우상단 "X/10" 형식 |
| 브랜드명 | "ONE STEP AHEAD" 우하단 (커버 슬라이드만) |

### 2-4. 컬러 팔레트
| 요소 | 색상 |
|------|------|
| 배경 사진 | 흑백 (그레이스케일) |
| 메인 텍스트 | 순수 흰색 (#FFFFFF) |
| 상단 레이블 | 흰색 (투명도 있을 수 있음) |
| 브랜드명 | 흰색 또는 밝은 회색 |
| 보조 색상 | 없음 (무채색 계열만 사용) |

→ **컬러 전략**: 흑백 사진 + 흰 텍스트만 사용 = 절제된 럭셔리감, 감성 잡지 무드

## 3. 슬라이드별 구성 패턴

### 커버 (1/10)
- 강렬한 흑백 인물 클로즈업
- 대주제 제목 중앙 하단 배치 (예: "사회적 성숙")
- 소제목 한 줄 추가 ("자신과 세계를 균형 있게 바라보며 삶을 세우는 능력")
- 브랜드명 "ONE STEP AHEAD" 우하단

### 본문 슬라이드 (2~9/10)
- 상단 소주제 레이블: "SOCIAL MATURITY" (영문 캡스)
- 번호 표시: "X/10" 우상단
- 슬라이드마다 독립적인 소주제 (예: "사회적 성숙", "타인의 내면을 읽는 예민함")
- 소주제 관련 인물 사진 (각 슬라이드마다 다른 사진)
- 제목 Bold 대형 + 본문 2~4줄 Regular 소형

### 마지막 슬라이드 (10/10)
- 텍스트 중심 (인용구 또는 종합 메시지)
- 사진 위에 긴 문장 오버레이
- 명언/철학적 문장으로 여운 남김

## 4. 콘텐츠 구성 패턴

### 시리즈 주제 구조 (사회적 성숙 예시)
```
커버: 대주제 선언 ("사회적 성숙")
  ↓
P2: 사회적 성숙 (개념 정의)
P3: 타인의 내면을 읽는 예민함
P4: 감정을 관조하며 다스리는 힘
P5: 결과를 감당하는 심리적 무게
P6: 갈등 속에서 피어나는 지혜
P7: 자기 성찰의 무한한 여정
P8: 공동체 속에서 실현되는 성장
P9: 겸손과 자존의 미묘한 조화
P10: 마무리 인용구 + 여운
```

→ **패턴**: 대주제를 하위 덕목/개념으로 분해 → 각 슬라이드가 하나의 에세이처럼 독립

### 텍스트 톤
- 딱딱하지 않음, 사유하는 문체
- 단정적 선언보다 "~이다", "~하는 것이다" 설명형
- 2~4줄로 완결되는 짧은 통찰
- 예시: "사회적 성숙함은 단순히 나이 먹거나 나 혼자의 능력이 아니라, 그것은 인간과 세계를 깊이 이해하고, 내면의 균형 속에서 책임과 선택을 정교하게 조율하는 지적·정신적 성숙이다."

## 5. Monologue 디자인에 적용할 스펙

### 현재 디자인과 레퍼런스 비교
| 항목 | 현재 Monologue | 레퍼런스 (@onestepahead) |
|------|---------------|------------------------|
| 배경 | Gemini 생성 컬러 이미지 | 흑백 빈티지 인물 사진 |
| 텍스트 위치 | 하단 40% 블랙 영역 | 사진 위 오버레이 (중하단) |
| 레이아웃 | 이미지 60% + 텍스트 40% 분리 | 풀블리드 + 텍스트 오버레이 |
| 컬러 | 블랙/골드 + 컬러 이미지 | 흑백 무채색 계열 |
| 분위기 | 모던 다크 | 빈티지 필름 감성 |

### 레퍼런스 스타일 적용 시 변경 포인트
1. **배경**: Gemini로 흑백 빈티지 감성 사진 생성 (컬러 X)
2. **레이아웃**: 상단 레이블 + 풀블리드 이미지 + 텍스트 오버레이 방식
3. **텍스트 위치**: 사진 중하단에 제목 Bold 배치, 바로 아래 본문 2~3줄
4. **브랜드**: "MONOLOGUE" 브랜드명 우하단 (커버), 상단에 페이지 번호
5. **컬러**: 흑백 이미지 + 흰 텍스트 (골드 강조선 최소화 또는 유지)

### Gemini 이미지 프롬프트 방향 (레퍼런스 스타일)

```
contemplative Korean professional woman in 30s,
1960s artistic portrait style,
soft grain texture, high contrast shadows,
introspective mood, no text,
seen from side or back,
cinematic still photography"
```
