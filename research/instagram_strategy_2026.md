## 1. 알고리즘 핵심 변화 (2025~2026)

### 가장 중요한 신호 순위
| 순위 | 신호 | 설명 |
|------|------|------|
| 1 | **저장(Save)** | "다시 보고 싶은 콘텐츠" = 알고리즘이 가장 높게 평가 |
| 2 | **공유(Share)** | DM 공유, 스토리 공유 모두 포함 |
| 3 | **체류 시간** | 슬라이드를 오래 볼수록 + "더 보기" 클릭 |
| 4 | **댓글** | 단순 이모지보다 텍스트 댓글 가중치 높음 |
| 5 | **좋아요** | 중요하나 예전보다 가중치 하락 |

### 핵심 패러다임 변화
- **해시태그 → 캡션 키워드 SEO** 로 발견(Discovery) 방식 전환 (2024년 말부터)
- 해시태그 팔로우 기능 종료됨 → 해시태그는 이제 "카테고리 분류" 역할만
- Instagram이 **텍스트/캡션/댓글까지 AI로 분석**해 관련 콘텐츠 추천

## 2. 해시태그 전략

### 결론: 3~5개, 정밀 타겟
- ❌ 과거: 30개 도배 → 현재는 오히려 노출 감소 페널티
- ✅ 현재: **3~5개** 고도로 관련성 높은 태그

### Monologue 계정 권장 해시태그 믹스 (5개 고정)
```
① 브랜드 태그 (1개):    #모노로그
② 대형 타겟 태그 (1개): #직장인  (검색량 100만+)
③ 주제 태그 (1개):      #번아웃 | #직장인번아웃 | #퇴근후
④ 철학/감성 태그 (1개): #의미치료 | #스토아 | #마음챙김
⑤ 날씨/시즌 태그 (1개): 오늘 날씨/계절에 맞게 동적 생성
                          예: #봄날씨 #월요일아침 #비오는날
```

### 위치: 캡션 본문 끝, 공백 줄 뒤에 배치
```
본문 텍스트

#모노로그 #직장인 #번아웃 #의미치료 #봄날씨
```

## 3. 캡션 구조 전략

### 핵심 원칙
- **첫 125자** = 피드에서 "더 보기" 없이 보이는 구간 → 가장 중요
- 첫 줄에 **주요 키워드** 자연스럽게 포함 (알고리즘 인식)
- **저장 유도 CTA** 필수 포함 (저장 = 1순위 신호)
- 캡션 길이: 본문 **150자 이내** (핵심만), 해시태그 포함 총 250자 이내 권장

### 검증된 캡션 템플릿 (Monologue용)
```
[첫 줄 훅 — 슬라이드 Hook 문구와 연결, 125자 이내]
오늘도 버텨낸 것만으로 충분합니다.

[본문 — 2~3줄, 핵심 키워드 포함]
야근 후 지하철, 그 무감각함의 이름을 오늘은 알아줬으면 해서요.
프랭클이 말한 '의미를 찾는 힘'은 거창한 게 아닙니다.

[저장 CTA]
📌 저장해두고 오늘 밤 꺼내 읽어보세요.

[댓글 유도 질문 — 선택]
오늘 하루 가장 힘들었던 순간은 언제였나요? 댓글로 나눠봐요.

[공백 줄]

#모노로그 #직장인 #번아웃 #의미치료 #봄날씨
```

### 금지 사항
- ❌ "좋아요 눌러주세요" → 효과 없음, 알고리즘 페널티 가능
- ❌ 해시태그 10개 이상
- ❌ 관련 없는 대형 태그 (#daily #love 등) 어뷰징

## 4. 콘텐츠 포맷 성과 비교

| 포맷 | 평균 참여율 | 저장률 | 도달 |
|------|-----------|--------|------|
| **카루셀(카드뉴스)** | **1.92%** | **3.4%** | 중상 |
| Reels | 0.50% | 1.2% | 최상 |
| 단일 이미지 | 0.45% | 0.8% | 하 |

→ **카루셀이 저장률 1위** = Monologue 방향 완전히 옳음

### 카루셀 최적 설정
- 슬라이드 수: **5~10장** (현재 5장 = 적정)
- 크기: 1080x1080 (정방형) 또는 1080x1350 (세로형이 화면 점유율↑)
- 마지막 슬라이드: 반드시 **CTA** 포함

## 5. 최적 포스팅 시간 (한국 직장인 타겟)

### 요일별 우선순위
```
1순위: 화, 수, 목 (직장인 루틴 안정기)
2순위: 금 (주말 앞두고 감성 소비 증가)
3순위: 월 (출근 피로감 — 공감 콘텐츠 반응 좋음)
피하기: 토, 일 (직장인 계정은 평일이 항상 우위)
```

### 시간대별 우선순위 (KST 기준)
```
🥇 오후 7시 ~ 10시   → 퇴근 후 소파/침대에서 스크롤 (최고 황금 시간)
🥈 오전 7시 ~ 9시    → 출근길 지하철/버스 (Monologue 타겟과 완벽 매칭)
🥉 오후 12시 ~ 1시   → 점심 시간
```

### Monologue 권장 포스팅 스케줄
```
화/수/목 오후 7시 30분  ← 1순위
월/금    오전 7시 30분  ← 퇴근길 공감 극대화
```

## 6. 캡션 SEO 키워드 전략

### Instagram 검색에서 발견되기 위한 키워드
캡션 첫 1~2문장에 자연스럽게 삽입할 키워드 목록:

```
[고검색량 타겟 키워드]
- 직장인, 번아웃, 퇴근, 야근, 직장생활
- 멘탈, 마음챙김, 위로, 공감

[중간 검색량 — 발견 가능성 높음]
- 의미치료, 스토아철학, 로고테라피
- 직장인번아웃, 직장인일상, 퇴근후일상

[롱테일 키워드 — 경쟁 낮고 정확 타겟]
- 야근하는직장인, 번아웃극복, 퇴근후혼자있고싶을때
```

## 7. 저장/공유를 유도하는 콘텐츠 패턴

### 저장률을 높이는 콘텐츠 유형
1. **나중에 실천할 것** — "오늘 해봐야지" 하는 Action 콘텐츠
2. **공감 캡처** — "이거 내 얘기다" → 친구 DM 공유
3. **인용구/명언** — 저장해서 배경화면으로 쓰고 싶은 것
4. **체크리스트/요약** — 정보 밀도 높은 것

→ Monologue의 5슬라이드 구조는 이 4가지를 모두 충족

### CTA 문구 순위 (참여율 기준)
```
1위: "저장해두고 오늘 밤 꺼내 읽어보세요 🤍"
2위: "이 말이 필요한 사람에게 공유해주세요"
3위: "오늘 하루 어땠나요? 댓글로 나눠요"
4위: "다음 편도 보고 싶다면 팔로우 🖤"
```

## 8. 적용 계획 (psychological_engine.py 수정 방향)

### 캡션 프롬프트 개선 포인트
```python
instagram_caption 생성 지시:
1. 첫 줄 = 슬라이드 Hook 문구와 연결된 강렬한 1문장 (키워드 포함)
2. 본문 = 2~3줄, 오늘의 날씨/상황 반영, 감성적 키워드 자연 삽입
3. 저장 CTA = "저장해두고 오늘 밤 꺼내 읽어보세요 🤍" (고정)
4. 댓글 유도 = 개방형 질문 1개
5. 공백 줄
6. 해시태그 5개 (형식 고정)

해시태그 생성 규칙:
- 반드시 5개
- #모노로그 (고정)
- #직장인 (고정)
- 오늘 뉴스/주제 관련 1개
- 감성/철학 관련 1개
- 오늘 날씨/계절/요일 관련 1개
```
