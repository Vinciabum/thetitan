# [PRD] Monologue: 3040 직장인을 위한 미니멀 심리 처방 서비스

## 1. 제품 개요 (Product Overview)
*   **서비스명:** Monologue (모노로그)
*   **슬로건:** "당신의 소란스러운 하루 끝, 고요한 독백의 시간"
*   **핵심 가치:** 3040 직장인이 처한 거시적 상황(뉴스, 날씨, 트렌드)을 심리학적 통찰(의미치료 & 스토아 철학)과 결합하여, 따뜻한 위로와 실질적인 인생의 '피벗(Pivot)' 포인트를 제안함.

## 2. 타겟 페르소나 (Target Audience)
*   **주요 대상:** 30-49세 남녀 직장인.
*   **심리 상태:** 번아웃, 커리어 정체기, 새로운 환경에 대한 갈망, 관계의 피로감.
*   **선호 스타일:** 화려함보다는 절제된 미니멀리즘, 감정적 호소보다는 깊이 있는 통찰.

## 3. 콘텐츠 구조 (5-Page Slide Algorithm)

| 페이지 | 역할 | 상세 내용 |
| :--- | :--- | :--- |
| **P1. Hook** | 질문 | 본질을 꿰뚫는 짧은 질문 (예: "오늘 당신의 이름은 '직급' 뒤에 숨어있지 않았나요?") |
| **P2. Context** | 연결 | 뉴스(사회적 온도) + 날씨(자연의 흐름) + 트렌드를 섞어 현재의 공기를 묘사 |
| **P3. Insight** | 통찰 | 빅터 프랭클의 의미치료와 스토아 철학을 결합한 따뜻하고 지적인 조언 |
| **P4. Action** | 변화 | 새로운 환경(물리적/심리적)을 향한 아주 작고 구체적인 오늘의 행동 지침 |
| **P5. Outro** | 소통 | AI 생성 소통 질문 + 팔로우/저장 유도 (CTA) |

## 4. 기술적 요구사항 (Technical Specifications)

### 4.1 데이터 수집 (Context Engine)
*   **Multi-Source Collector:** 
    *   `GNews`: 최신 경제/사회 뉴스 키워드 추출.
    *   `Weather API`: 현재 날씨/시즌 데이터 연동.
    *   `Social Trends`: 대중의 심리적 키워드(불안, 희망, 휴식 등) 분석.

### 4.2 AI 페르소나 (Psychological Engine)
*   **Role:** "빅터 프랭클의 철학을 계승한 20년 경력의 시니어 커리어 코치"
*   **Tone of Voice:** 따뜻하고 신뢰감 있는 1인칭 관찰자 시점. 부드러운 공감형 말투.

### 4.3 비주얼 가이드라인 (Visual Engine)
*   **Style:** 고요한 일상 사진 (B타입). 빈 의자, 창가의 햇살, 새벽 거리 등 미니멀리즘 이미지.
*   **Typography:** 산세리프 계열 모던 폰트, 여백을 극대화한 레이아웃.

## 5. 단계별 로드맵 (Roadmap)

### Phase 1: 기반 구축 (Current)
*   기존 AutoGram 엔진을 Monologue 페르소나로 전환.
*   미니멀리즘 이미지 생성 프롬프트 최적화.

### Phase 2: 플랫폼 확장 (Short-term)
*   **Threads**: 텍스트 중심의 빠른 배포 로직 추가.
*   **Blog (Naver/Google)**: 슬라이드 내용을 장문의 포스팅으로 변환하는 서술형 엔진 개발.

### Phase 3: 개인화 처방 (Mid-term)
*   사용자 인풋(DM, 댓글) 기반의 1:1 맞춤형 심리 처방 카드 생성 기능.

### Phase 4: 생태계 구축 (Long-term)
*   정기 구독형 심리 레터 및 커뮤니티 연동.
